# 🎮 Valorant Role Atlas (Full Stack)

Uma aplicação Full Stack responsiva e interativa que consome dados de personagens do jogo Valorant. O projeto permite buscar agentes específicos ou filtrar por classes (Controlador, Iniciador, Sentinela, Duelista), apresentando uma interface premium inspirada no design original do jogo.

## ✨ Funcionalidades
* **Busca Inteligente:** Algoritmo de busca que entende plurais (ex: "Iniciador" ou "Iniciadores") e nomes compostos (ex: "KAY/O").
* **Efeitos Visuais Premium:** Animações CSS em 3D (Card Flip), transições suaves e layout dinâmico usando Grid e Flexbox.
* **Sistema de Seleção Lateral:** Exibição modular onde o usuário pode navegar por agentes da mesma classe em um menu lateral sem fechar o card principal.
* **API Própria (Backend):** Servidor Node.js utilizando Express para roteamento e entrega de dados no formato JSON.

## 🛠️ Tecnologias Utilizadas
* **Frontend:** HTML5, CSS3 Avançado (Animações, Flexbox, Grid), JavaScript Vanilla (Fetch API, DOM Manipulation).
* **Backend:** Node.js, Express.js.
* **Arquitetura:** Separação estrita de responsabilidades (HTML, CSS e JS em arquivos isolados) e rotas RESTful no backend.

## 🚀 Como executar o projeto localmente

1. Clone o repositório:
\`\`\`bash
git clone https://github.com/SEU-USUARIO/Valorant-Role-Atlas.git
\`\`\`

2. Instale as dependências do Node:
\`\`\`bash
npm install
\`\`\`

3. Inicie o servidor:
\`\`\`bash
node server.js
\`\`\`

4. Abra no navegador:
\`\`\`text
http://localhost:3000
\`\`\`
